## 🚀 Featured Projects
 
### ☕ Connect'In Java
**Stack:** Java, Spring Boot, Docker, Maven, Spring Security, JUnit
 
Full migration of a Laravel backend to Spring Boot, built as a pair project at Epitech Paris. Developed a Java REST API with JWT authentication, handling users, posts, comments, likes, and teams, documented with OpenAPI/Swagger.
 
### 🎮 JeuxVidéOps
**Stack:** JavaScript, GitHub Actions, Docker, GitHub Pages, Node.js
 
Complete CI/CD pipeline for retro JavaScript games. Automated unit and functional testing, linting with the Google JavaScript Style Guide, and dependency security audits, all running on GitHub Actions.
 
### 🎬 My Cinema
**Stack:** PHP, MySQL, HTML, CSS, JavaScript
 
Admin dashboard for cinema management, built solo. Hands-on practice of OOP, MVC architecture, CRUD operations, database relations, and PDO prepared statements for query security.
 
### 🌱 CarbonLint (in progress)
**Stack:** Static analysis, CI/CD integration
 
Open-source static analysis tool that measures the energy footprint of code directly in CI/CD pipelines. My contribution to greener software, born from my Linux Foundation Green Software certification.
